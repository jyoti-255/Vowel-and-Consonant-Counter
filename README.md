# 🅰️🅱️ Vowel and Consonant Counter 🎉

This Django application allows users to enter a text and get the count of vowels and consonants present in the text. ✨

## 🌟 Features

- 📝 Enter a text string.
- 🔢 Calculate the number of vowels and consonants.
- 📊 Display the result.
